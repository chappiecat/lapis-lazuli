# lapis-lazuli

## サイト概要
ハンドメイドアクセサリー限定で売買するフリマサイト

### サイトテーマ
趣味でハンドメイドアクセサリーを売買出来、趣味を副収入に出来るフリマサイト

### テーマを選んだ理由
アクセサリーの売買を通して、コロナ渦で希薄になっている人と人とのつながりを結びなおす事が目的で作りました

### ターゲットユーザー
- 普段使いのアクセサリーを探している人
- 製作したアクセサリーを売買したい人
- 製作したアクセサリーの評価を知りたい人
- 冠婚葬祭などの人生のイベントに身に着けるアクセサリーを探している人
- 大切な人への贈り物や言葉を伝えたい人

### 主な利用シーン
- 普段使いのアクセサリーを探している時
- 製作したアクセサリーを売買したい時
- 製作したアクセサリーの評価を知りたい時
- 冠婚葬祭などの人生のイベントに身に着けるアクセサリーを探している時
- 大切な人への贈り物や言葉を伝えたい時
